## 📏 间距变量

所有间距值基于 8px 的基础单位，形成一致的间距系统。

| 变量名 | 值 | 用途 | 使用场景 |
|--------|-----|------|---------|
| `$spacing-xs` | `4px` | 极小间距 | 紧凑元素间距 |
| `$spacing-sm` | `8px` | 小间距 | 内部间距、小元素间距 |
| `$spacing-md` | `16px` | 中间距 | 标准间距、卡片内边距 |
| `$spacing-lg` | `24px` | 大间距 | 区块间距、大元素间距 |
| `$spacing-xl` | `32px` | 超大间距 | 主要区块间距 |
| `$spacing-2xl` | `48px` | 特大间距 | 页面主要分区间距 |
| `$spacing-3xl` | `64px` | 极大间距 | 页面顶部/底部间距 |

**使用示例**：
```scss
.card {
  padding: $spacing-md;        // 16px 内边距
  margin-bottom: $spacing-lg;  // 24px 下边距
  gap: $spacing-sm;            // 8px 间隙
}
```

## 🔤 字体族变量

### 正文字体

```scss
$font-family-base: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 
                   'Hiragino Sans GB', 'Microsoft YaHei', '微软雅黑', 
                   'Helvetica Neue', Helvetica, Arial, sans-serif;
```

**用途**：所有正文、段落、描述文字

**使用示例**：
```scss
body {
  font-family: $font-family-base;
}

p {
  font-family: $font-family-base;
}
```

### 标题字体

```scss
$font-family-heading: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 
                      '微软雅黑', sans-serif;
```

**用途**：标题、大型文字、强调内容

**使用示例**：
```scss
h1, h2, h3 {
  font-family: $font-family-heading;
}

.title {
  font-family: $font-family-heading;
}
```

### 代码字体

```scss
$font-family-mono: 'SF Mono', 'Monaco', 'Inconsolata', 'Roboto Mono', 
                   'Source Code Pro', 'Consolas', monospace;
```

**用途**：代码块、代码片段、技术文本

**使用示例**：
```scss
code {
  font-family: $font-family-mono;
}

pre {
  font-family: $font-family-mono;
}

.code-block {
  font-family: $font-family-mono;
}
```

## 📏 行高变量

| 变量名 | 值 | 用途 | 使用场景 |
|--------|-----|------|---------|
| `$line-height-tight` | `1.2` | 紧凑行高 | 标题、代码、数字 |
| `$line-height-normal` | `1.5` | 正常行高 | 正文、段落、列表 |
| `$line-height-relaxed` | `1.8` | 宽松行高 | 长文本、描述、引用 |

**使用示例**：
```scss
h1 {
  line-height: $line-height-tight;  // 1.2 - 紧凑
}

p {
  line-height: $line-height-normal;  // 1.5 - 正常
}

.description {
  line-height: $line-height-relaxed; // 1.8 - 宽松
}
```

**选择指南**：
- **1.2（紧凑）**：用于标题、数字、代码，需要紧凑视觉
- **1.5（正常）**：用于正文、段落、列表，标准可读性
- **1.8（宽松）**：用于长文本、描述、引用，增强可读性

## 📊 Z-index 层级变量

Z-index 层级系统用于管理浮层元素的堆叠顺序。

| 变量名 | 值 | 用途 | 使用场景 |
|--------|-----|------|---------|
| `$z-dropdown` | `1000` | 下拉菜单 | 下拉菜单、下拉列表 |
| `$z-sticky` | `1020` | 粘性元素 | 粘性导航、粘性头部 |
| `$z-fixed` | `1030` | 固定定位元素 | 固定导航、固定按钮 |
| `$z-modal-backdrop` | `1040` | 模态框背景 | 模态框遮罩 |
| `$z-modal` | `1050` | 模态框 | 对话框、弹窗 |
| `$z-popover` | `1060` | 弹出框 | 气泡提示、弹出菜单 |
| `$z-tooltip` | `1070` | 提示框 | 悬停提示、通知 |

**堆叠顺序**（从下到上）：
```
z-index: 1000  ← 下拉菜单
z-index: 1020  ← 粘性元素
z-index: 1030  ← 固定定位元素
z-index: 1040  ← 模态框背景
z-index: 1050  ← 模态框
z-index: 1060  ← 弹出框
z-index: 1070  ← 提示框（最上层）
```

**使用示例**：
```scss
.header {
  z-index: $z-fixed;  // 固定导航
}

.modal {
  z-index: $z-modal;  // 模态框
}

.notification {
  z-index: $z-tooltip;  // 通知提示
}
```

# 📝 使用规范

## 颜色使用规范

**✅ 正确做法**：
```scss
.button {
  background-color: $primary-color;  // 使用变量
  color: white;
  border: 1px solid $border-color;
}
```

**❌ 错误做法**：
```scss
.button {
  background-color: #E41E2B;  // 硬编码颜色
  color: white;
  border: 1px solid #E5E5E5;  // 硬编码颜色
}
```

## 间距使用规范

**✅ 正确做法**：
```scss
.card {
  padding: $spacing-md;
  margin-bottom: $spacing-lg;
  gap: $spacing-sm;
}
```

**❌ 错误做法**：
```scss
.card {
  padding: 16px;        // 硬编码间距
  margin-bottom: 24px;  // 硬编码间距
  gap: 8px;             // 硬编码间距
}
```

**✅ 正确做法**：

body {
  font-family: $font-family-base;

}

h1 {
  font-family: $font-family-heading;

## 🎯 注意事项

1. **始终使用变量**：尽量不要硬编码设计值
2. **保持一致性**：同类型元素使用相同的变量
3. **遵循层级**：使用合适的 z-index 避免堆叠冲突
4. **响应式设计**：使用断点变量实现响应式布局
5. **添加注释**：在使用变量时添加说明注释
6. **定期审查**：定期检查是否有新的硬编码值需要变量化
