





# 开发说明——李凤祥

-`OrganizationTree.vue` :用来**全局管理**，导入.json数据并将根目录数据传递给`TreeNode` 开始逐步渲染递归结构。

-`TreeNode.vue` :通过自己调用自己逐步**生成递归结构**，节点的展开/收起直接在本组件内部完成，查看详情信息需要由`OrganizationTree` 调用`NodeDetail` 完成。

-`NodeDetail.vue` :详情卡片，管理成员信息、负责人信息、统计信息的**卡片样式结构**。

-`organization.json` :可维护的.json文件，存放或修改人员信息。

# 运行说明

`fronted/src/components`:存放三个`.vue`文件；

`fronted/src/Data`:Data文件夹为自建文件夹，用来存放`.json` 文件。

修改`src/router/index.js`:为树状图页面配置路由

~~~html
 //第20行
const OrganizationTree = () => import('../components/OrganizationTree.vue')

.......

//第148行
 {
    path: '/organization', // 访问路径，可自定义为 '/team'、'/structure' 等
    name: 'OrganizationTree', // 路由名称，用于编程式导航
    component: OrganizationTree, // 使用上面懒加载导入的组件
    meta: { title: '组织架构 - 华创智库' } // 务必设置标题，路由守卫会用它更新页面标题
  },
~~~



修改`src/components/The Header`:在主导航栏添加链接

~~~html
//第200行
{
    id: 7, // id 保持唯一，顺序递增即可
    title: '组织架构', // 导航栏显示的文字
    path: '/organization', // 必须与你在路由文件中配置的 `path` 完全一致
    icon: 'sitemap' // 建议图标，表示组织结构。如果不确定，可先用 'users' 或 'diagram-project'
  }
~~~
